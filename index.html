Estrutura básica do HTML

Antes de colocar a mão na massa é importante criar uma pasta dedicada para seu projeto, para seja importada para sua IDE, no caso do VScode criamos um arquivo com nome de index.html.
Devido a boas praticas, esse nome é o padrão para página inicial de seu site.

    Agora vamos de fato a estrutura
  • Html: É o elemento raiz que define todo o documento HTML
  • Head: Sua finalidade é definir o corpo do documento
  • Body: Usada como cabeçalho, ou seja, um conjunto de informações de uma página de um site

Sobre TAGS !

Já iniciamos falando sobre as TAGS, ao mencionar a estrutura básica do HTML. Mas afinal, o que são TAGS ?
Tags são estruturas que contém instruções para que seu navegador possa interpretar uma pagina web.

As principais são:
1. <title><title/> - Identifica sua página web.
2. <p> <p/> - Principal tag de texto, compõe um parágrafo.
3. <img> - Não necessita de fechamento. Utilizada para inserir uma imagem.
4. <ol> <ol/> - Para criação de uma lista ordenada.
5. <u> <ul/> - Para criação de uma lista não ordenada.

Para saber mais pesquise pelo site MDN Web Docs - Mozilla

Listas Ordenadas e não ordenadas

Listas ordenadas: são usadas para indicar alguma sequência ou numeração. As tags usadas para criar uma lista ordenada são <ol> e <li>. A tag <ol> é usada para definir a lista e a tag <li> é usada para cada item da lista.

Listas não ordenadas: são usadas para listar itens, sem se preocupar com sua sequência. Chamamos de lista de marcadores apenas. As tags usadas para criar uma lista não ordenada são <ul> e <li>. A tag <ul> é usada para definir a lista e a tag <li> é usada para cada item da lista.

Implementando Links

O link é a função que permite inserir os hiperlinks. Para incluir um link utilizamos a tag <a> e dentro dessa tag incluimos o atributo href. Veja no exempo abaixo:


<href="url">Exemplo</a>
